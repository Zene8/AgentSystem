# Run Test Suite

## Description
Execute full test suite and report coverage.

## Steps
1. Run tests: `npm test` (or equivalent for your stack)
2. Wait for completion
3. Report: total tests, passed, failed, skipped
4. Show coverage percentage
5. If any fail, list failures with error messages

## Trigger
User runs: `/run-tests`

## Output
Test results summary. If failures, show diff of what broke.
